# Custom 2D Game Engine (C++)
Пет-проект собственного легковесного 2D-игрового движка, написанного на **C++** с нуля. Проект создавался с целью глубокого изучения низкоуровневой архитектуры игровых систем, паттернов проектирования, ручного управления памятью и работы с графическими API.  

Модульный 2D-игровой движок, написанный на **C++** с использованием мультимедийной библиотеки **SFML**. Движок проектировался как независимая библиотека (статическая/динамическая) с готовым игровым циклом, подсистемами анимации, коллизий и сущностей. 

Для удобства старта разработки внутри репозитория предусмотрен скрипт автоматического создания шаблона новой игры на базе скомпилированного движка.

---

## Архитектура и ключевые фичи

*   **Архитектурное разделение:** Сам движок собирается в отдельную библиотеку (`build/libSFML_project.a`), изолируя системную логику от кода конкретной игры.
*   **Игровой цикл (Game Loop):** Встроенный игровой цикл, все логика запуска и инициализации всех элементов движка уже в самой библиотеке.
*   **Система сущностей (Actors):** Базовый класс `Actor`, предоставляющий интерфейс для создания игровых объектов, обновления их состояния и отрисовки.
*   **Модуль коллизий (Collision System):** Собственная легковесная система расчета и разрешения столкновений объектов.
*   **Система анимаций (Animation System):** Готовый пайплайн для работы с покадровой анимацией и переключением состояний персонажей.
*   **Developer Experience (DX):** Наличие скрипта-генератора для мгновенного развертывания нового проекта под разработку игры на базе движка.
